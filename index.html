Derivada por Definición: Explicación Detallada

1. Definición

La derivada de f(x) está definida como:

f′(x) = limh→0 [f(x + h) − f(x)] / h

2. Ejemplo: f(x) = x²

Aplicamos la definición:

( (x + h)² − x² ) / h
= (x² + 2xh + h² − x²) / h
= (2xh + h²) / h
= 2x + h → (al límite h→0) → 2x

Por lo tanto, f′(x) = 2x.

3. Estructura del proceso

  • Escribir el límite: limh→0 [f(x + h) − f(x)] / h
  • Reemplazar f(x + h)
  • Expandir y simplificar
  • Cancelar h y tomar el límite
  • Obtener la derivada
